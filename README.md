# will-yergin-site

A fast, minimal Next.js + Tailwind personal site optimized for recruiters and hiring managers.

**Live:** https://willjyergin.com

## Local dev (Windows 11 / macOS / Linux)

1. Install Node.js (LTS recommended).
2. In this folder:

   npm install
   npm run dev

Open http://localhost:3000

## Deploy (recommended)

This site is deployed on Vercel. Pushing to `main` triggers an automatic deployment.

- Vercel: import the GitHub repo and deploy. Default settings work.

(Optional alternative: Cloudflare Pages with the Next.js preset.)
